AnyItemConvertible

public protocol AnyItemConvertible

A type which can be converted into a AnyItem, so you do not need to explicitly wrap / convert your ItemContent in an Item when providing it to a Section.

Section("id") {
    MyItemContent(text: "Hello, World!")
}

struct MyItemContent : ItemContent {
   var text : String
   ...
}

Only two types conform to this protocol:

Item

The Item conformance simply returns self.

ItemContent

The ItemContent conformance returns Item(self), utilizing the default values from the Item initializer.
