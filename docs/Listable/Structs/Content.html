  • callAsFunction(_:configure:)

    Allows streamlined creation of sections when building a list, leveraging Swift’s callAsFunction feature, allowing treating objects as function calls.

    In layperson’s terms, this allows you to replace code like this:

    listView.configure { list in
    list += Section("section-id") { section in
        ...
    }
}

    With this code, which is functionally identical:

    listView.configure { list in
    list("section-id") { section in
        ...
    }
}

    Swift
