FlowAppearance

public struct FlowAppearance : ListLayoutAppearance

Allows rendering a list in the style of a flow layout. Items in the list flow from one row to the next, with each row containing as many items as will fit. Items can be the same sizes or different sizes.

You can control the layout both via the FlowAppearance parameter passed to .flow layout types, plus via the section.layouts.flow options on a Section.

To display a flow layout in your list, set its layout to a .flow type:

list.layout = .flow { flow in
    // Customize the flow options.
}
